## What Is EngageAI

This platform enables business owners to:

- Set up their business profile to personalize messaging style.

- Add customers along with their specific pain points and interaction history.

- Receive a personalized 4-step SMS roadmap generated by OpenAI.

- Review, edit, and approve each message before sending.

- Automatically schedule and deliver SMS messages precisely via Celery and Twilio.
